1. HTML

Los comandos HTML más básicos incluyen etiquetas de estructura que definen el esqueleto de una página web.

Etiquetas principales de la estructura del HTML
# Etiqueta Función
1 <html> Delimita el inicio y el final del código HTML.
2 <head> Contiene metadatos sobre la página.
3 <body> Define el contenido principal y visible de la página.
